ABSTRACT:

Este projeto consistiu na elaboração de um diário digital. Esta aplicação suporta múltiplos utilizadores. Cada utilizador pode publicar vários tipos de conteudos, como por exemplo, registos desportivos, fotos, receitas, entre outros. Estas publicações podem ser privadas ou públicas conforme a vontade do utilizador, por defeito estas são privadas. Os utilizadores desta plataforma têm acesso a todas as suas publicações e a todos os conteúdos públicos.

Esta aplicação foi desenvolvida recorrendo à framework Express, sendo que foi utilizado JavaScript e para o seu frontend foi utilizado o motor de templates PUG. Relativamente à base de dados foi usado MongoDB, sendo esta aplicação orientada a documentos.

Para a elaboração deste projeto foram utilizadas técnicas e soluções abordadas nas aulas. Além disso foi necessário recorrer a mecanismos não abordados nas aulas com intuito de cumprir todos os requisitos. Com isto não só praticamos o que foi lecionado nas aulas, como aprendemos novas e diferentes soluções.
